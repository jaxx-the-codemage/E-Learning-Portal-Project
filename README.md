# E-Learning Portal Project

## About
This project is a web-based E-Learning Portal that i developed for my Web-development Module Assignment.

## Features
- User registration
- Student dashboard
- Instructor dashboard
- Course upload and management
- Material upload (PDF files)
- Update and delete courses
- Contact page

## Technologies Used
- HTML
- PHP
- MySQL
- CSS

## Database Setup
1. Open MySQL Workbench (localhost:3306)
2. Create a new schema / database
3. Open the `eportal.sql` file
4. Copy and paste the SQL code into a query tab
5. Run the script to create the tables and data

## How to Run
1. Copy the project folder into `xampp/htdocs`
2. Start Apache in XAMPP
3. Make sure MySQL server is running
4. Import `eportal.sql` in MySQL Workbench
5. Open the project in browser using localhost
